## 实现多种语言上传商品并翻译的电商平台

## 项目简介
实现上传八种语言其中的一种，可以翻译成另外八种语言，点击查看后台[Swagger](http://39.105.138.38:8080/mq/swagger-ui.html)

### 运用知识
- 使用MySQL数据库查询和插入数据、Druid连接池
- java Web三层框架
- 采用Spring+SpringMVC+MyBatis开发项目
- 使用Maven对该项目进行管理
- 使用Swagger对该项目的后台进行可视化，达到前后端分离

## 准备
 1.在pom.xml中导入第三方jar包
 2.建立包结构
 3.编写代码
 4.测试代码
 
 ## 搭建环境
 eclipse + MySQL
